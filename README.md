# Rà soát hợp đồng

Tên hiển thị: Rà soát hợp đồng

Tên kỹ thuật: `rasoathopdong`

`rasoathopdong` là ứng dụng desktop Windows hỗ trợ rà soát hợp đồng local/offline theo bộ tiêu chí pháp lý có cấu trúc.

Nguyên tắc dữ liệu:

- Không gửi hợp đồng ra ngoài.
- Không analytics.
- Không telemetry.

Trạng thái hiện tại: Giai đoạn 0 - Khởi tạo nền.

Repo đang ở bước khởi tạo solution/project. Chưa có engine rà soát, chưa có WebView2 UI, chưa có SQLite.
